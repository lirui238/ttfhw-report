遇到的问题 (4)

方案: 安装unzip包：apt-get install -y unzip

来源: cmake/third_party/json.cmake错误信息

UT测试段错误：Segmentation fault，getenv BUILD_PATH failed

方案: 设置环境变量：export BUILD_PATH=/workspace/build

来源: tests/ut/op_host/test_op_host_main.cpp源码分析

算子包安装失败：缺少libes_math.so运行态依赖

方案: 使用--force参数跳过验证继续安装（仅用于编译态验证）

来源: 安装脚本提示信息

方案: 无法解决，需要完整运行态环境（CANN ops包+驱动固件）

来源: 链接器错误信息，文档说明编译态vs运行态区别
