遇到的问题 (4)

Docker Hub 拉取 ubuntu:24.04 镜像失败（连接重置）

方案: 本地已有缓存镜像 ubuntu:24.04 (c4a8d5503dfb)，直接使用本地镜像

来源: docker images 检查

Meson setup 时 Lua 依赖未找到

方案: Lua 是可选依赖，meson.build 中已处理：'Lua not found, Lua modules will not be built' 和 'Lua tests will be skipped'，不影响构建

来源: meson.build 第97行 dbus_dep 附近 + src/luaclib/meson.build 第139行

securec 依赖未找到

方案: securec 是可选特性，mcbase 使用内建实现回退。'mcbase 使用内建 securec'

来源: libraries/mcbase/meson.build 第42-48行 + meson_options.txt securec option

module_test_example 样例程序在所有测试通过后退出时发生段错误 (SIGSEGV)

方案: 程序正常功能测试全部通过 (EXIT CODE 0)，段错误发生在退出时的清理阶段。可能是静态/共享库卸载顺序或全局对象析构问题。此问题不影响构建和测试验证结果。

来源: builddir/examples/module_test/module_test_example 运行时观察
