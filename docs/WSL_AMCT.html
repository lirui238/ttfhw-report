Missing securec.h header file during build

根因: securec.h is part of the base CANN toolkit which must be installed separately at ASCEND_HOME_PATH

来源: CMakeLists.txt references ASCEND_HOME_PATH for CANN dependencies

PyTorch installation timeout

根因: Network connectivity issues to PyPI from container environment

来源: N/A - network issue

Git safe.directory issue with cmake FetchContent

根因: Git security feature blocking cmake FetchContent operations

来源: CMake FetchContent module error message

根因: AMCT build requires full CANN toolkit installation, not just AMCT-specific packages
