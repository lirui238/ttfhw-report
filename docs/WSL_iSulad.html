conclusion
iSulad 在 Ubuntu 24.04 环境下构建被阻塞。主要原因是：1) Ubuntu 标准 lxc 包不包含 iSulad 需要的特定 API 扩展；2) 从源码构建带补丁的 lxc 过程复杂；3) openEuler 容器网络问题导致无法使用包管理器。建议在 openEuler 系统上进行构建验证。
