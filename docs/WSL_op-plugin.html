遇到的问题 (1)

Build failure: 'CLOCK_MONOTONIC_RAW' was not declared in this scope

根因: Header file ordering issue - <time.h> not included before macro usage in newer GCC

方案: export CXXFLAGS='-include time.h' to force time.h inclusion before compilation

来源: GCC 13.3 stricter header requirements
