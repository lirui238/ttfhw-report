遇到的问题 (5)

openEuler 官方 repo (repo.openeuler.org) SSL 握手超时

方案: 切换到华为云镜像源 mirrors.huaweicloud.com/openeuler/

来源: 自行排查网络连通性后发现华为云镜像可用

ubs-comm-devel 包不在 SP2 仓库中

方案: 按 docs/build_install/构建指导.md 2.2.2 节的指引,从源码构建 ubs-comm

来源: docs/build_install/构建指导.md - 2.2.2 其他openEuler版本依赖安装说明

dnf builddep 命令不可用

方案: 安装 dnf-plugins-core 包提供 builddep 插件

来源: dnf 错误提示信息

挂载路径问题 (Docker Desktop + Git Bash on Windows)

方案: 使用 Windows 风格绝对路径 'D:\code\ttfhw_3\ubs-engine\tmp_src' 进行卷挂载

来源: Docker 错误信息排查
