遇到的问题 (5)

ASCConfig.cmake not found during cmake configuration

方案: Downloaded and installed CANN toolkit package (Ascend-cann-toolkit_9.0.0_linux-x86_64.run, 1.3GB) from Huawei cloud mirror

来源: docs/zh/install/quick_install.md mentions CANN toolkit installation requirement

Permission denied on .run file execution

方案: chmod +x /tmp/Ascend-cann-toolkit_9.0.0_linux-x86_64.run before running

来源: Standard permission fix

packaging Python module not found during build

方案: pip3 install packaging --break-system-packages

来源: Build script dependency, not listed in requirements.txt

Build takes ~97 minutes to compile flash_attention_score operator

方案: This is expected due to kernel compilation for multiple tiling configurations using bisheng compiler

来源: NPU kernel compilation is inherently time-consuming

Sample execution requires NPU hardware

方案: Cannot be resolved in Docker container without NPU device. Requires physical Ascend NPU or CANN Simulator

来源: docs/zh/install/quick_install.md - runtime dependencies section
