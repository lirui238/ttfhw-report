方案: 切换到 Ubuntu 24.04 x86_64 镜像

方案: 使用 GIT_SSL_NO_VERIFY=true 绕过 TLS 问题

来源: TLS 包解码错误

pip install -e . 失败

方案: 使用 python -m build 构建 wheel 包，通过 --no-deps 参数安装成功

来源: 依赖 pandas/torch 等版本兼容问题

方案: 无法解决 - 需要 Ascend NPU 硬件

来源: MindSpeed-MM 是昇腾 NPU 专用框架
