遇到的问题 (5)

ubs-virt-ovs RPM build failed: missing libasan-static dependency

方案: Installed libasan-static via yum (dependency found in ubs-virt-ovs.spec BuildRequires)

来源: ubs-virt-ovs/build/ubs-virt-ovs.spec

virt-optimizer build failed: ubs-opt-tuner compilation error (std::shared_ptr not found)

方案: Root cause: ebpf/src/optimizer/util/ui_manager.h uses std::shared_ptr but missing #include <memory>. This is an upstream code bug.

来源: Source code analysis - ui_manager.h

virt-awaresched: 2 security manager tests fail (testModifyEffectiveCapabilities, testClearCapabilities)

方案: Tests return error code 14 vs expected 0. Likely WSL2 kernel limitation for process capability manipulation. Expected to pass on native openEuler.

来源: Test output analysis - WSL environment limitation

ubs-virt-enpu: cannot build (requires ASCEND_HOME_PATH environment variable)

方案: This component requires Ascend NPU SDK which is not available on x86 WSL. It is a hardware-specific component for Ascend NPU platforms.

来源: ubs-virt-enpu/vcann-rt/build_ut.sh line 5

virt-awaresched vas_daemon: cannot create /var/log/vas/ directory for logging

方案: Permission denied in WSL environment. Expected to work on native openEuler installation with proper permissions. Binary otherwise functional.

来源: Binary execution output
