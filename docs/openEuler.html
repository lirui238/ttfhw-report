issues_and_solutions
  1. #1
    4 个字段
    issue
    configure 脚本找不到 Boot JDK
    cause
    容器中初始只安装了 JDK 11，但 OpenJDK 8 构建需要 JDK 8 作为 Boot JDK（需要 rt.jar）
    solution
    安装 java-1.8.0-openjdk-devel 包，使用 --with-boot-jdk=/usr/lib/jvm/java-1.8.0-openjdk 指定正确的路径
    status
    已解决
  2. #2
    4 个字段
    issue
    缺少 cmp 工具
    cause
    openEuler 基础镜像未安装 coreutils 包
    solution
    安装 coreutils 和 diffutils 包
    status
    已解决
  3. #3
    4 个字段
    issue
    缺少 fontconfig-devel
    cause
    OpenJDK 构建需要 fontconfig 库支持字体渲染
    solution
    安装 fontconfig-devel 包（同时自动安装了 libxml2-devel、gettext 等依赖）
    status
    已解决
  4. #4
    4 个字段
    issue
    容器挂载路径错误
    cause
    初始容器挂载到了错误的目录（test 目录而非 bishengjdk-8 源码目录）
    solution
    重新创建容器，正确挂载 /root/<user>/Documents/<path>/ttfhw_bisheng/tmp_src 到 /workspace
    status
    已解决
