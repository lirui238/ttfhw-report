遇到的问题 (3)

Python 依赖安装时清华镜像源连接超时

方案: 切换使用阿里云镜像源 (https://mirrors.aliyun.com/pypi/simple/)

来源: 实际测试发现

构建时间较长，容器会话可能中断

方案: 建议使用 screen 或 tmux 保持会话，或使用 nohup 后台运行

来源: 实际测试发现

CANN 环境为必需依赖，必须安装后才能编译

方案: 使用 scripts/init_env.sh 脚本自动安装 CANN 环境

来源: docs/build.md + scripts/init_env.sh
